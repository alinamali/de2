# 2. Сконфигурируйте файловое хранилище на сервере HQ-SRV:
------------------------------------------------------------------------------------
### 2.1 При помощи двух подключенных к серверу дополнительных дисков размером 1 Гб сконфигурируйте дисковый массив уровня 5
### 2.2 Имя устройства – md0, при необходимости конфигурация массива размещается в файле /etc/mdadm.conf
### 2.3 Создайте раздел, отформатируйте раздел, в качестве файловой системы используйте ext4
### 2.4 Обеспечьте автоматическое монтирование в папку /raid
------------------------------------------------------------------------------------

#### *Настройка hq-srv*

mdadm --create -verbose /dev/md0 -l 5 -n 2 /dev/vdb /dev/vdc

В данной команде смщтрим уровень - это -l,

если допустим уровень 0, то команда будет выглядеть след. образом

mdadm --create -verbose /dev/md0 -l 0 -n 2 /dev/vdb /dev/vdc

------------------------------------------------------------------------------------
```bash
apt-get update && apt-get install mdadm
lsblk 
mdadm --zero-superblock --force /dev/vdb /dev/vdc
mdadm --create --verbose /dev/md0 -l 5 -n 2 /dev/vdb /dev/vdc
lsblk 
mdadm --detail --scan >> /etc/mdadm.conf
mkfs.ext4 /dev/md0
mkdir /raid
cp /etc/fstab /etc/fstab.back
echo "/dev/md0 /raid ext4 defaults 0 0" >> /etc/fstab
mount -av
df -T
```

#  3. Настройте сервер сетевой файловой системы (nfs) на HQ-SRV:
------------------------------------------------------------------------------------
### 3.1 В качестве папки общего доступа выберите /raid/nfs, доступ для чтения и записи исключительно для сети в сторону HQ-CLI
### 3.2 На HQ-CLI настройте автомонтирование в папку /mnt/nfs
### 3.3 Основные параметры сервера отметьте в отчёте
------------------------------------------------------------------------------------
*Настройка hq-srv*
------------------------------------------------------------------------------------
```bash
mkdir /raid
df -h 
apt-get install -y nfs-server nfs-utils
mkdir /raid/nfs
chmod 777 /raid/nfs


ip-address - vlan 200 подсеть смотреть на HQ-CLI


echo "/raid/nfs    192.168.1.0/27(rw,no_subtree_check,no_root_squash)" >> /etc/exports
exportfs -arv
systemctl enable --now nfs-server

```

------------------------------------------------------------------------------------
*Настройка hq-cli*
------------------------------------------------------------------------------------
```bash
apt-get update && apt-get install nfs-utils nfs-clients
mkdir /mnt/nfs
chmod 777 /mnt/nfs
(fstab)


ip-address - смотреть на HQ-SRV


echo "192.168.0.10:/raid/nfs    /mnt/nfs    nfs defaults    0    0" >> /etc/fstab
mount -av 
df -h


ls -l /raid/nfs  (на hq-srv)
```
