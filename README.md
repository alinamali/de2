# 4. Настройте службу сетевого времени на базе сервиса chrony на маршрутизаторе ISP:
------------------------------------------------------------------------------------
### 4.1 Вышестоящий сервер ntp на маршрутизаторе ISP - на выбор участника
### 4.2 Стратум сервера - 5
### 4.3 В качестве клиентов ntp настройте: HQ-SRV, HQ-CLI, BR-RTR, BR-SRV.
------------------------------------------------------------------------------------
*Настройка isp*
------------------------------------------------------------------------------------
```bash
(в файле /etc/chrony.conf прописываем след. после строчки: pool pool.ntp......, не забыть ее еще закоментировать!!!!


server ntp0.ntp-servers.net iburst prefer mistratum 4
local stratum 5
allow 0.0.0.0/0

systemctl restart network
systemctl restart chronyd
```
------------------------------------------------------------------------------------
 *Настройка br-rtr*
------------------------------------------------------------------------------------
```bash
en
conf t
ntp server 172.16.2.1
wr mem
do sh ntp status
```
------------------------------------------------------------------------------------
*Настройка hq-srv, hq-cli, br-srv*
------------------------------------------------------------------------------------
```bash
apt-get update && apt-get install chrony -y
echo "server 172.16.1.1(or 172.16.2.1) iburst" >> /etc/chrony.conf
systemctl restart chronyd
chronyc sources
```
